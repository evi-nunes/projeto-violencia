# Você não está sozinha

Site informativo de apoio a mulheres em situação de violência doméstica, com foco em Fortaleza/CE.

## 🎯 Objetivo

Oferecer informação acessível e acolhedora sobre como reconhecer a violência, conhecer direitos, buscar ajuda e encontrar serviços de apoio. Todos os serviços, endereços e contatos indicados são referentes à cidade de Fortaleza, Ceará.

## 📄 Seções

- Como reconhecer violência doméstica
- Tipos de violência
- Seus direitos
- Como buscar ajuda
- Para onde ligar
- Onde se amparar

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
